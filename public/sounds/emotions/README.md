# 🎵 Zvukovi za igru Emocija

Ovaj folder sadrži MP3 zvukove koji se koriste u igri **"Moja Osećanja"**.

## 📋 Potrebni fajlovi

Ubaci sledeće MP3 fajlove u ovaj folder:

### 🎭 Zvukovi emocija (opcioni - za svaku emociju)
| Fajl | Opis |
|------|------|
| `happy.mp3` | Vesela melodija / smeh / zvuk sreće |
| `sad.mp3` | Tužna melodija / plakanje |
| `angry.mp3` | Zvuk ljutnje / uznemirenosti |
| `scared.mp3` | Zvuk straha / napetosti |

### 📖 Zvukovi scenarija (pitanja koja se čitaju)
| Fajl | Opis |
|------|------|
| `scenario_1.mp3` | "Kako se osećaš kada te neko pohvali i kaže da si odličan?" |
| `scenario_2.mp3` | "Kako se osećaš kada ti drug uzme igračku?" |
| `scenario_3.mp3` | "Kako se osećaš kada dobiješ poklon?" |
| `scenario_4.mp3` | "Kako se osećaš kada padneš i udariš koleno?" |
| `scenario_5.mp3` | "Kako se osećaš kada ti neko sruši kulu od kocaka?" |
| `scenario_6.mp3` | "Kako se osećaš kada se izgubiš u prodavnici?" |
| `scenario_7.mp3` | "Kako se osećaš kada neko želi da se igra sa tobom?" |
| `scenario_8.mp3` | "Kako se osećaš kada te juri strašan pas?" |

### ✅ Zvukovi povratne informacije
| Fajl | Opis |
|------|------|
| `correct.mp3` | Zvuk za tačan odgovor (može biti isti kao `/sounds/success.mp3`) |
| `wrong.mp3` | Zvuk za netačan odgovor / ohrabrenje |
| `encouragement.mp3` | "Pokušaj ponovo, ti to možeš!" |

## 📝 Napomene
- Svi fajlovi treba da budu u `.mp3` formatu
- Preporučena dužina zvukova: **2-5 sekundi** za emocije, **5-10 sekundi** za scenarije
- Fajlovi se učitavaju direktno iz browsera, nema potrebe za AI generisanjem
